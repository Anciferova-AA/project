# Проект: Сайт школы иностранных языков
- Описание проекта: Веб-сайт представляет собой платформу для онлайн-записи на курсы иностранных языков, предоставления информации о преподавателях, расписании занятий, ценах и доступных языках.  Ресурс включает в себя каталог курсов, личный кабинет для учеников с доступом к материалам для обучения и домашним заданиям.
